# 🌸 Study Blossom Queen 👑✨

A cute pink aesthetic student productivity web app built with pure HTML, CSS, and JavaScript 💖

Designed to help students stay organized, focused, hydrated, and motivated — all in one beautiful app.

---

## 💎 Features

✨ Login system (saved locally)  
👤 Custom profile picture upload  
🌙 Dark / Light mode toggle  
🌷 Mood tracker  
🎯 Daily goal tracker  
✅ To-Do list (auto saved)  
📅 Weekly planner (auto saved)  
🌊 Water intake tracker  
🔥 Study streak counter  
⏰ Pomodoro timer with session counter  
🧠 Flashcards system  
🎧 Lofi & Rain study sounds  
✨ Animated sparkles & floating bubbles  
📱 Mobile app-style design  
💾 Automatic local storage saving  

---

## 🛠 Built With

- HTML5
- CSS3 (custom styling & animations)
- Vanilla JavaScript
- LocalStorage API

No frameworks. No backend. 100% front-end.
